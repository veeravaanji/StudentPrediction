Student Performance Prediction System

This project predicts whether a student will PASS or FAIL using a Machine Learning model.

Technologies Used

  • Python
  • Flask
  • Random Forest Classifier
  • Scikit-Learn
  • Pandas
  • SQLite Database
  • Bootstrap 5

Input Features

  • Student Name
  • Student ID
  • Study Hours
  • Attendance
  • Past Exam Score
  • Internet Access
  • Parent Education
  • Extracurricular Activities
